Групове середнє 1

\[ \bar{x}_{1}= \frac{1}{n} \sum\limits_{i=1}^{k}{x_{i} n_{i}} = {{ mean1 }} \]

Групове середнє 2

\[ \bar{x}_{2}= \frac{1}{n} \sum\limits_{i=1}^{k}{x_{i} n_{i}} = {{ mean2 }} \]

Загальне групове середнє

\[ \bar{x} = {{ common_mean }} \]

Групова дисперсія 1

\[ D_{1гр} = \frac{1}{n}\sum\limits_{i=1}^{k}{n_{i}(x_{i}-\bar{x}_{в})^{2}}= {{ pvariance1 }} \]

Групова дисперсія 2

\[ D_{2гр} = \frac{1}{n}\sum\limits_{i=1}^{k}{n_{i}(x_{i}-\bar{x}_{в})^{2}}= {{ pvariance2 }} \]

Внутрішньогрупова дисперсія

\[ D_{внгр} = \frac{1}{n}\sum N_{j} D_{jгр}= {{ inner_variance }} \]

Міжгрупова дисперсія

\[ D_{міжгр} = \frac{1}{n}\sum N_{j} (\bar{x}_{j} - \bar{x})^{2}= {{ intergroup_variance }} \]

Загальна дисперсія

\[ D_{заг} = \frac{1}{n}\sum n_{j} (\bar{x}_{j} - \bar{x})^{2}= {{ common_variance }} \]
